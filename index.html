Key Features

🔍 Autodiscovery & Registry

OpenDex automatically scans your network, identifies MCP servers, and populates your Skills DB with version-controlled schemas.

💾 Semantic Outcome Layer

Cache successful tool executions in vector format and serve known outcomes in under 50ms.

🛡️ Enterprise Segregation

Strict isolation between community tools and enterprise systems using mTLS, role policies and PII scrubbing.

📊 Outcome Interface

Convert raw JSON tool outputs into Markdown summaries and interactive UI widgets for HITL workflows.
